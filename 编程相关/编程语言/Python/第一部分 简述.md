# Python教程

- 以Python3.6为基础
- 本教程假定读者有用==`C/C++`或`Java`==的编程经验
- ==`./SampleCode`==为示例代码



## 安装环境

### Windows：
- 下载页地址 `https://www.python.org/downloads/windows/`
- 下载地址 `https://www.python.org/ftp/python/3.6.7/python-3.6.7-amd64.exe`
